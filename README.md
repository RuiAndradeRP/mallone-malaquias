# Mallone Malaquias

Sistema web desenvolvido em PHP para gestão de serviços de uma barbearia.

## Funcionalidades

- Área pública
- Gestão de serviços
- Área administrativa protegida por login
- Adicionar, editar e apagar serviços
- Ligação à base de dados MySQL

## Tecnologias

- PHP
- MySQL
- HTML5
- CSS3
- PDO
- XAMPP
