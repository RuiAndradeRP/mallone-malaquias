# Mallone Malaquias - Sistema de Gestão para Barbearia

Projeto desenvolvido em PHP e MySQL para a gestão de serviços da barbearia Mallone Malaquias.

## Funcionalidades

- Página inicial personalizada
- Gestão dinâmica de serviços
- Área administrativa protegida por login
- Adicionar, editar e remover serviços
- Sistema de autenticação com sessões
- Integração com base de dados MySQL

## Tecnologias Utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- PDO
- XAMPP
- Git e GitHub

## Objetivo

Desenvolver uma solução web para apoiar a gestão de uma barbearia, proporcionando uma interface moderna e intuitiva para administradores e clientes.
